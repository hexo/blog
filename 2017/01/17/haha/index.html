发表于 2017-01-17 | 分类于 python |

最近由于实验室开发项目需要，两个项目需要不同的Python环境，所以研究了一下pyenv和virtualenv这两个工具的使用方法，下面简要介绍一下

pyenv

pyenv是一个能简易地在多个Python版本中进行切换的工具，它简单而优雅

  • 进行全局的Python版本切换
  • 为单个项目提供对应的Python版本
  • 只依赖Python本身
  • 自带virtualenv 能够进行virtualenv管理
  • 提供下载不同的Python版本和衍生版本
